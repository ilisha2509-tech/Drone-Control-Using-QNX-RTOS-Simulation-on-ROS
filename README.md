# Drone Control Using QNX RTOS & ROS

## Overview
This project focuses on the development and simulation of a drone control system using QNX Real-Time Operating System (RTOS) integrated with Robot Operating System (ROS).

The system was designed for real-time drone control, autonomous robotics research and simulation-based testing.

---

## Features
- Real-time drone control using QNX RTOS
- ROS-based simulation environment
- Autonomous navigation concepts
- Sensor and actuator integration
- Robotics-oriented system architecture

---

## Technologies Used
- QNX RTOS
- ROS (Robot Operating System)
- Embedded Systems
- Robotics Simulation
- Linux Environment

---

## Applications
- Autonomous drones
- Robotics research
- Real-time embedded systems
- Simulation and testing

---

## Project Highlights
- Integration of RTOS with robotics middleware
- Simulation-driven drone testing
- Real-time system architecture for drone applications
